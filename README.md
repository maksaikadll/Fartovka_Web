# Сайт игры Фартовка

Современный веб-сайт для игры Фартовка.

## Описание

Этот сайт представляет игру Фартовка с современным дизайномs. Сайт включает:

- **Hero секция** с анимированным фоном и призывом к действию
- **О игре** - информация об игре и статистика
- **Особенности** - ключевые функции игры
- **Типы карт** - описание всех типов карт
- **Правила игры** - интерактивные вкладки с правилами
- **Скачать** - информация об установке и системных требованиях

## Структура файлов

```
Web/
├── index.html      # Главная HTML страница
├── styles.css      # Стили в стиле Riot Games/Valorant
├── script.js       # JavaScript для интерактивности
└── README.md       # Этот файл
```

## Запуск

### Локальный запуск

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (если установлен http-server)
npx http-server
```

3. Откройте в браузере: `http://localhost:8000`

### Использование с веб-сервером

Просто скопируйте файлы в директорию вашего веб-сервера (Apache, Nginx и т.д.)

## Особенности дизайна

- **Анимации**: Плавные переходы и эффекты при наведении
- **Адаптивность**: Полностью адаптивный дизайн для мобильных устройств
- **Интерактивность**: Анимированные счетчики, вкладки, эффекты ripple
- **Типографика**: Современный шрифт Inter от Google Fonts

## Кастомизация

### Изменение цветов

В файле `styles.css` измените CSS переменные в `:root`:

```css
:root {
    --primary-color: #ff4655;      /* Основной цвет */
    --secondary-color: #0f1923;    /* Фоновый цвет */
    --text-primary: #ece8e1;       /* Основной текст */
    /* и т.д. */
}
```

### Добавление контента

Отредактируйте `index.html` для добавления новых секций или изменения контента.
